# mach.lang.target.isa.x64

x86\_64 instruction set. Implements
[`isa.IsaVTable`](../isa.md#isavtable) and registers itself at
compiler startup with [`isa.register`](../isa.md#register).

Source is `new/lang/target/isa/x64.mach` (currently empty).

## Identifying constants

- `id` = [`isa.ARCH_X86_64`](../isa.md#constants).
- `name` = `"x86_64"`.
- `pointer_width` = 8 bytes.
- `endianness` = [`target.ENDIAN_LITTLE`](../../target.md#constants).

## Register classes

| Class   | Bit width | Count | Notes                                                                  |
|---------|-----------|-------|------------------------------------------------------------------------|
| `gpr`   | 64        | 16    | RAX, RCX, RDX, RBX, RSP, RBP, RSI, RDI, R8–R15.                        |
| `xmm`   | 128       | 16    | XMM0–XMM15 (SSE; AVX-256 / AVX-512 not exposed in this class).         |
| `flags` | 64        | 1     | EFLAGS / RFLAGS, modelled as a single register for clobber tracking.   |

`rsp` and `rbp` are reserved by the [`frame`](../../be/codegen/frame.md)
pass; the allocator must not see them as general candidates.

## Instruction encoding

The encoder lives in [`be.codegen.encode`](../../be/codegen/encode.md);
this vtable does not own emission. What this module *does* own is
the canonical mapping from
[`MIR`](../../be/codegen/mir.md) opcodes to per-arch instruction
forms, exposed as part of the
[isel](../../be/codegen/isel.md) dispatch table.
