# Operators

## Unary

```
!   # not
+   # positive
-   # negative
?   # reference
@   # dereference
```

## Binary

```
=   # assignment
==  # equality
!=  # inequality
>=  # greater than or equal to
<=  # less than or equal to
>   # greater than
<   # less than
+   # addition
-   # subtraction
*   # multiplication
/   # division
**  # exponentiation
%   # modulus
&&  # logical and
||  # logical or
```

## Binary (bitwise)

```
&   # and
|   # or
^   # xor
~   # not
<<  # left shift
>>  # right shift
```

## Other Symbols

```
.   # member access

*   # pointer type

,   # argument separator
;   # statement separator
```
