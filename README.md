# MACH

Mach is a low level, compiled, statically typed programming langauge.

# Key features

- No magic. No side effects. No bullshit.
- Small, clearly defined, and well documented feature set.
- Familiar and easy to read syntax. You should not need to take a class to use mach.
- Barebones standard library with enough to do the basics, but not enough to overload the language.

# Credit

Much inspiration for the design and actual implementation of this language was taken from other projects. They are listed below:
